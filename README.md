FactChecker App

Overview

FactChecker App is a Streamlit-based web application that allows users to upload PDF documents and automatically verify claims using live web data.

Features

* Upload PDF files
* Extract claims from PDF
* Verify information from the web
* Show results as Verified, Inaccurate, or False

Technologies Used

* Python
* Streamlit
* PyPDF2
* DuckDuckGo Search
* Pandas

Deployment

The application is deployed on Streamlit Cloud.
